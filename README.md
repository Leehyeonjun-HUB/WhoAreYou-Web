# WhoAreYou-Web
MiniProject-애착유형 자가진단 사이트

# WhoAreYou이란?
### -> 공식 애착유형 테스트를 그대로 가져와서 내 자신의 유형을 파악할 수 있는 웹사이트
 
**1. 팀 구성** <br>
  * 이현준, 이정원

**2. Architecture** <br>
  * MVC Pattern
  
**3. Skill Set** <br>
  * Front-end : Javascript, JSP, HTML, CSS, Bootstrap    
  * Back-end : Java    
  * DB : Oracle 
  * Data Source : 애착유형검사 공식 홈페이지
  * Library : Lombok, json-simple   
  * Environment : Eclipse (Maven)
